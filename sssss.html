css引入方式

方式一：

<link rel="stylesheet" href="CSS/test.CSS">

方式二：
<!--在html中使用-->
<style Type="text/css">
    @import url(myStyle.css);   /*这里是通过@import引用CSS的样式内容*/
</style>

<!--在css中使用-->
@import url("public-head-footer.css");

Flex布局

行内元素也可以使用Flex布局

.box{
    display:inline-flex;

注意：Flex布局以后，子元素的float、clear和vertical-align属性将失效。

六个容器属性设置

.box{
    display:flex;
    display: -webkit-flex; /*Webkit内核的浏览器,必须加上-webkit前缀*/
    flex-direction:row | row-reverse | column | column-reverse;
    /*基本的排列方向主要就分为竖向， 横向*/

    flex-wrap: nowrap  |   wrap   |   wrap-reverse;
    /*不换行，  换行， 在第一行的上方换行*/

    flex-flow: <flex-direction> || <flex-wrap>;    
    /*上面两个属性的合写*/

    justify-content: flex-start | flex-end | center | space-between | space-around;
        /*box里面内容的横向对齐方式:    左对齐,   右对齐,   中对齐,   两端对齐,    每个内容间距相同对齐*/

    align-items:flex-start   |  flex-end   |   center  |  baseline  |stretch
    /*box里面的内容的竖向对齐方式: 上对齐  | 下对齐  |  中对齐  |  以第一行文字的基线为准对齐  |  顶天顶地的对齐*/

    align-content: flex-start   |   flex-end   |  center   |space-between |  space-around | stretch
    /*多根轴线的对齐方式:轴起点对齐,    轴终点对齐,  轴中点对齐,  轴左右间距相同,  轴上下左右间距都相同,    轴线占满*/
}

六个项目属性

.iten{
    order:  整数;      /*数值越小,越靠前*/

    flex-grow:比例;   /*默认是0,如果都是1,就会等分*/
    flex-shrink:缩小比例;  /*默认是1,0是不能缩放*/

    flex-basis:表示占据了主要空间;   /*默认值auto*/

    flex:  auto none;    /*flex-grow, flex-shrink 和 flex-basis的简写*/

    align-self:  auto | flex-start | flex-end | center | baseline | stretch;   /*允许不一样的对齐方式*/
}

选择器

伪类选择器/伪元素选择器

  • 伪类选择器：E:nth-child(n)
  • 伪元素选择：E::selection
  • ”:” 与 “::” 区别在于区分伪类和伪元素

    注意：E:after、E:before在旧版本里是伪类，在新版本里是伪元素，新版本下E:after、E:before会被自动识别为E::after、E::before，按伪元素来对待,我们一定要按照规范书写。

IE和W3C盒模型

;

IE模型下 width = padding + content + border;
W3C盒模型下 width = content

- IE盒模型只会出现在IE5版本和IE6+的怪异模式中。
- CSS3中可以通过box-sizing 来指定盒模型，即可指定为content-box、border-box;
box-sizing: border-box;  //width = border + padding + content
box-sizing: content-box;  //width = content

css3案例

案例一：悬停

效果：当鼠标悬停的时候，自动出现阴影，和文字


<div class="item" data-brief="整圆">
    <div class="border-radius"></div>
</div>

.item {
    width: 210px;
    height: 210px;
    position: relative;
}

.item::after {
    content: attr(data-brief);
    display: block;
    width: 100%;
    height: 100%;
    text-align: center;
    line-height: 210px;
    position: absolute;
    top: 0;
    left: 0;
    background-color: rgba(170, 170, 170, 0);
    z-index: -1;
    transition: all 0.3s ease-in;
}

.item:hover::after {
    background-color: rgba(170, 170, 170, 0.6);
    z-index: 100;
}
